### nodemon
For monitor any changes in your source and automatically restart your server.

### bcryptjs
Password encryption

### express-validator
Data validation via 'express'

### jsonwebtoken
Generating an authorization token

### shortid
Creates amazingly short non-sequential url-friendly unique ids
